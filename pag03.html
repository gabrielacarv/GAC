Projeto de Startup


Ideia inicial

Criar o bot para o WhatsApp no intuito de divulgar os preços dos combustíveis dos postos da cidade, facilitando a escolha dos motoristas para economizar na hora de abastecer.

Funcionamento:

A ideia inicial é que através do WhatsApp o usuário possa mandar uma mensagem para o bot solicitando a relação de preço dos combustíveis da cidade, ou que ele se cadastre para receber diariamente essas informações. Os dados coletados com os postos seriam tratados e enviados de forma acessível para o usuário, com indicação do posto com o preço mais atrativo e outras 3 opções de estabelecimentos. "Como seria feita a coleta de dados?" A coleta seria feita a partir de parcerias com os postos, onde eles nos enviaram os valores dos combustíveis. "E o que eles ganham ao fornecer esses dados?" A divulgação dos seus preços na nossa rede de contatos, possivelmente atraindo clientes caso a faixa de preço seja conveniente ao consumidor. A ideia é ampliar futuramente para um aplicativo que aumentaria as funcionalidades com integração para outros apps, como os de mapa.

Trabalho de Extensão


O projeto para o TDE - (Trabalho de Extensão), teve como tema: “Iniciação da lógica de programação para crianças do ensino fundamental”. Realizamos pesquisas de ferramentas e softwares para auxiliar e complementar o ensino para as crianças. A aplicação web escolhida pelo grupo foi o Blockly.


Blockly Games

O que é Blockly Games ?

É uma série de jogos educativos que ensinam programação. É projetado para crianças que não tiveram experiência prévia com programação de computadores. Ao final desses jogos, os jogadores estão prontos para usar linguagens convencionais baseadas em texto.
